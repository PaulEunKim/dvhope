##### How to setup
1. Access this URL on your browser: https://www.python.org/downloads/release/python-3119/
2. Under Files, choose the installer for your system
3. Run the installer (Don't forget to check "Add python.exe to PATH")
4. Restart your PC.
5. Verify the installation by running the command `python --version` on your terminal (the version should be shown if the install successfully finished)
6. Run the command `pip install --no-cache-dir xgboost==2.1.2 shap==0.46.0 matplotlib==3.9.2 plotly==5.24.1`(it may take time)
7. Run the command `cd {your path to run global.py}`(it could take time)
8. Run the command `python global.py`(the application automatically run)
